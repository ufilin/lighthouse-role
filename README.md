# Ansible роль: lighthouse-role

**License**: MIT

Роль для автоматической установки и настройки [Lighthouse](https://github.com/VKCOM/lighthouse) — инструмента для мониторинга и визуализации состояния серверов, разработанного VKCOM.

## Описание

Данная роль выполняет следующие действия:

- Настраивает DNS-сервер (8.8.8.8) в `/etc/resolv.conf`.
- Обновляет кэш пакетов APT.
- Устанавливает `nginx` и `git`.
- Удаляет предыдущую версию каталога Lighthouse (если существует).
- Создаёт каталог для Lighthouse (по умолчанию `/var/www/lighthouse`).
- Клонирует репозиторий Lighthouse (ветка `master`).
- Устанавливает корректные права владельца (`www-data`) на каталог Lighthouse.
- Удаляет стандартные конфигурации Nginx (default сайты).
- Генерирует конфигурацию Nginx для Lighthouse из шаблона `lighthouse.conf.j2`.
- Включает сайт Lighthouse.
- Проверяет корректность конфигурации Nginx.
- Перезапускает и включает Nginx в автозагрузку.

Роль предназначена для целевых систем на базе Debian/Ubuntu (используется модуль `apt`).

## Требования

- Ansible версии 2.9 или выше.
- Управляемый хост с ОС семейства Debian/Ubuntu.
- Доступ к интернету для установки пакетов и клонирования репозитория.
- Права `become: true` для установки пакетов, создания файлов и управления системными службами.

## Переменные

Роль использует следующие переменные (задаются в `defaults/main.yml` и `vars/main.yml`):

| Имя переменной        | Значение по умолчанию                     | Описание                                  |
|-----------------------|-------------------------------------------|-------------------------------------------|
| `lighthouse_dir`      | `"/var/www/lighthouse"`                   | Путь к каталогу установки Lighthouse      |
| `lighthouse_repo`     | `"https://github.com/VKCOM/lighthouse.git"` | URL репозитория Lighthouse               |
| `lighthouse_version`  | `"master"`                                | Ветка или тег репозитория для клонирования |
| `nginx_user`          | `"www-data"`                              | Пользователь Nginx                        |
| `nginx_group`         | `"www-data"`                              | Группа Nginx                              |

Эти переменные можно переопределить в playbook или инвентори.

## Шаблоны

Роль использует два шаблона Jinja2:

- **`lighthouse.conf.j2`** — конфигурационный файл Nginx для сайта Lighthouse.  
- **`nginx.conf.j2`** — основной конфигурационный файл Nginx (опционально).
